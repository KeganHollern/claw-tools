# Security Policy

## Supported Versions

Currently, only the latest `main` branch version of plugins is actively supported.

## Reporting a Vulnerability

We take the security of these tools very seriously. If you discover a security vulnerability in any of the plugins, please do not disclose it via a public GitHub issue.

Instead, please report it privately:
- Message Kegan directly on Telegram: [@lmaokegan](https://t.me/lmaokegan)
- Use contact methods listed at [https://lystic.dev](https://lystic.dev)

Please include the following information in your report:
- The type of vulnerability.
- A step-by-step description of how to reproduce the vulnerability.
- Any potential impact it might have.

You should receive a response within 48 hours. Thank you for helping keep this project safe!
